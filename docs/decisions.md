# Decisões técnicas do projeto

## Escolha de tecnologias

A opção por PWA foi feita por sua portabilidade e simplicidade inicial, permitindo o desenvolvimento com tecnologias web já conhecidas e facilitando a evolução futura do projeto.

## Evolução futura do projeto

A escolha por PWA não descarta uma possível migração futura para React Native.
Por esse motivo, o projeto busca manter a lógica de negócio separada da camada de interface, facilitando uma eventual reimplementação da UI em outra tecnologia.
